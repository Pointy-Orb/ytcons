# YTCons
The app that allows you to view Youtube videos from the terminal!

## Features:
* Feeds, (like youtube subscriptions) with folders for organizing subscriptions!
* Search
* Playlists
* Download as well as play videos (or just the audio, if desired)
* View channel pages!
